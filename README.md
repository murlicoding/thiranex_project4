README Highlights
Business Problem: Understand sales performance and identify opportunities to improve profitability.
Data Cleaning: Removed duplicates, handled missing values, and formatted date columns.
Analysis: Sales trends, regional performance, customer segments, and product profitability.
Visualizations: Histograms, bar charts, heatmaps, scatter plots, and line charts.
Key Findings:
Technology products generated the highest revenue.
Some heavily discounted products resulted in negative profits.
The West region produced the highest sales.
Corporate customers contributed significantly to revenue.
Future Work: Sales forecasting, customer segmentation, and recommendation systems.
